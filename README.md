# OC.Documenty.pl

Production-ready platform for generating Polish insurance documents.

## Tech Stack
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- shadcn/ui

## Features
- OC cancellation generator
- PDF preview
- Sign & Send workflow
- Responsive UI
- SEO-first architecture

## Roadmap
- Landing page
- Multi-step generator
- PDF engine
- User dashboard
- Future backend integration
